Simulador de Encriptación Estilo RSA

Los sistemas bancarios reales multiplican dos números primos secretos muy grandes para blindar información. Digita un dato confidencial para ver cómo se transforma utilizando claves primas.

La fortaleza de los primos

Cualquier computadora puede multiplicar dos números primos al instante. Sin embargo, si un atacante intercepta el resultado gigante, revertir la operación para hallar los factores primos originales toma tanto tiempo que el mensaje se vuelve matemáticamente inquebrantable.
